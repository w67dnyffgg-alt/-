心の種帳 v2 完全版

## 中身
- index.html
- style.css
- script.js

## 実装機能
- ホーム
- 感情メモ帳
- お気に入り
- 感情の矢印ライブラリ
- カスタムガチャ設定
- 感情の矢印ガチャ
- ガチャ履歴
- 4ページ漫画テンプレ
- 作品ステータス管理
- 完成作品アルバム
- 感情コレクション
- JSONバックアップ書き出し・読み込み

## 使い方
index.html をブラウザで開いてください。

## 注意
データはブラウザの localStorage に保存されます。
ブラウザデータを削除すると消える可能性があるため、定期的にJSONバックアップを書き出してください。
